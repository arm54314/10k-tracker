1. โซนความเหนื่อย/เพซ (E, M, T, I, R)

EEasy

เพซที่วิ่งแล้วยังคุยเป็นประโยคยาวๆ ได้สบาย หายใจไม่หอบ ใช้ในการซ้อมส่วนใหญ่ของแผน (วันธรรมดา วอร์มอัพ คูลดาวน์ วิ่งยาวส่วนใหญ่) สร้างระบบแอโรบิกโดยไม่เค้นร่างกาย ฟื้นตัวเร็ว

MMarathon / Steady pace

เร็วกว่า Easy แต่ยังคุมต่อเนื่องได้เป็นสิบๆ นาที "สบายแต่มีจุดหมาย" ใช้ท้ายวิ่งยาวเพื่อฝึกร่างกายให้คุ้นจังหวะใกล้เพซแข่งตอนขาล้า

TThreshold / Tempo

เพซ "หนักแบบสบายๆ" พูดได้แค่ประโยคสั้นๆ แต่คุมต่อเนื่องได้ 15-25+ นาที ฝึกกำจัดกรดแลคติกให้ดีขึ้น เป็นหัวใจของ tempo run

IInterval

เพซเร็วระดับ VO2max วิ่งต่อเนื่องได้ไม่นาน จึงแบ่งเป็นช่วงสั้นๆ สลับจ็อกฟื้นตัว ใช้ดันเพดานฟิตเนสสูงสุดในแผนช่วงท้าย

RRepetition

เร็วกว่า I อีก ใกล้เพซแข่งระยะสั้น เน้นความเร็ว/รูปแบบวิ่ง ใช้เบาๆ สัปดาห์สุดท้ายเพื่อ "ปลุกขา" ก่อนแข่ง

VDOTตัวเลขฟิตเนส

ตัวเลขประเมินฟิตเนสโดยรวมของ Jack Daniels คำนวณจากผลแข่ง/time trial ใช้ทำนายเพซ E/M/T/I/R และเวลาแข่งที่เหมาะกับฟิตเนสปัจจุบัน ยิ่งฟิตขึ้น VDOT ยิ่งสูงขึ้น

2. ประเภทของเซสชันซ้อม

Easy run

วิ่งที่เพซ E ล้วนๆ ตลอดระยะ สร้างไมล์สะสม (aerobic base) โดยไม่เค้นระบบมากเกินไป เป็นวันซ้อมส่วนใหญ่ในแผน

Long run

วิ่งระยะไกลที่สุดของสัปดาห์ ปกติเพซ E บางสัปดาห์มีท้ายรันเร่งเป็น M หรือเพซแข่งเพื่อจำลองความรู้สึกช่วงท้ายแข่งจริง

Tempo run

มีท่อนกลางเป็นเพซ T ต่อเนื่อง คั่นด้วยวอร์มอัพ-คูลดาวน์เพซ E ก่อน-หลัง ฝึกวิ่งเร็วแบบคุมจังหวะได้นาน

Cruise intervals

คล้าย tempo แต่แบ่งเพซ T เป็นช่วงสั้นๆ หลายรอบสลับจ็อกฟื้นตัว สะสมเวลาที่เพซ T ได้มากขึ้นโดยไม่ล้าเกินไป

Intervals (I-pace)

ท่อนวิ่งเร็วระดับเพซ I สลับจ็อกฟื้นตัว ดันเพดาน VO2max เป็นท่อนที่หนักที่สุดในแผน มีเฉพาะช่วงร่างกายพร้อมแล้ว

Race-pace pickup

ช่วงท้ายของ long run/tempo ที่ปรับเพซใกล้เพซแข่งจริง ให้ร่างกายและสมองคุ้นชินกับความรู้สึกวิ่งเพซแข่งล่วงหน้า

Strides

ท่อนเร่งสั้นๆ (15-25 วิ) แบบควบคุมได้ ทำหลัง Easy run จบแล้ว 4-6 เที่ยว ฝึกความเร็ว/running economy โดยไม่สร้างความล้าสะสม

Shakeout

วิ่งสั้นเบามาก (15-20 นาที) ก่อนแข่ง 1-2 วัน เพื่อคลายขา กระตุ้นระบบเล็กน้อยโดยไม่สร้างความล้าเพิ่ม

Time trial

วิ่งจับเวลาเต็มที่ในระยะที่กำหนด ใช้แทนแข่งจริงเพื่อวัดฟิตเนสปัจจุบัน แล้วคำนวณ VDOT ใหม่ ปรับเพซซ้อมให้ตรงร่างกายจริง

3. โครงสร้าง/ศัพท์เกี่ยวกับการวางแผน

WU / CD

Warm-up / Cool-down — วิ่งเบาเพซ E ก่อน (WU) และหลัง (CD) ท่อนซ้อมหนัก เตรียมกล้ามเนื้อ-ข้อต่อ และลดความเสี่ยงบาดเจ็บ

Cutback week

สัปดาห์ผ่อน ลดปริมาณ/ความหนักลง (ทุก ~4 สัปดาห์) ให้ร่างกายฟื้นตัวและซึมซับผลซ้อม ก่อนกลับไปเพิ่มโหลดต่อ

Taper

ลดปริมาณซ้อมมากในสัปดาห์สุดท้ายก่อนแข่ง ให้ร่างกายพักฟื้นเต็มที่ พร้อมใช้พลังงานเต็มที่ในวันแข่ง

กฎ 10%

เพิ่มระยะ/ปริมาณซ้อมรวมต่อสัปดาห์ไม่เกิน ~10% จากสัปดาห์ก่อน ให้กระดูก-เอ็น-กล้ามเนื้อปรับตัวทัน ลดความเสี่ยงบาดเจ็บ

Negative split

ทำเวลาครึ่งหลังของระยะทางเร็วกว่าครึ่งแรก กลยุทธ์แข่งที่ปลอดภัยและมักให้ผลดีกว่าสำหรับนักวิ่งมือใหม่

Cross-training

กิจกรรมออกกำลังกายชนิดอื่น (ในแผนนี้คือแบดมินตัน) เสริมสมรรถภาพหัวใจ-ปอดโดยใช้กล้ามเนื้อคนละแบบกับวิ่ง ลดความเสี่ยงบาดเจ็บจากการใช้ซ้ำ

4. สัญญาณเตือน & การฟังร่างกาย

HR (Heart Rate)

ใช้เช็คความหนักคู่กับเพซ วัน Easy ควรอยู่ระดับต่ำ-กลาง (ไม่เกิน ~150-155 bpm) ถ้าสูงกว่าปกติทั้งที่เพซเดิม อาจเป็นสัญญาณว่ายังฟื้นตัวไม่เต็มที่

Overtraining

สะสมความล้าจนไม่ทันฟื้น สังเกตจาก: นอนไม่ค่อยหลับ, ชีพจรพักสูงขึ้นผิดปกติหลายวันติด, เจ็บจุดเดิมซ้ำๆ, เหนื่อยง่ายกว่าปกติ — พบสัญญาณเหล่านี้ให้ลดโหลดทันที
